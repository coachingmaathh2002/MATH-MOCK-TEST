First Order DEs: Variable Separable Method Quiz

1. Solve the differential equation: \( \frac{dy}{dx} = \frac{x^2}{y^3} \).

2. Find the general solution of \( \frac{dy}{dx} = e^{x-y} \).

3. Solve the initial value problem: \( \frac{dy}{dx} = 2xy^2 \), with \( y(1) = 1 \).

4. Determine the general solution to \( (1+x^2)dy + (1+y^2)dx = 0 \).

5. Solve \( y' = \frac{\sin x}{\cos y} \).

6. Find the solution to \( \frac{dy}{dx} = \frac{x}{y\sqrt{1-x^2}} \).

7. Which of the following is a general solution to \( \frac{dy}{dx} = \frac{y \ln y}{x} \)?

8. Solve \( (x+1)\frac{dy}{dx} = xy^2 \).

9. Find the solution to \( \frac{dy}{dx} = \frac{\sqrt{y}}{x} \) with \( y(1) = 4 \).

10. Solve \( \frac{dy}{dx} = \frac{x^2+1}{y^2+1} \).

11. Find the implicit solution for \( \frac{dy}{dx} = \frac{y \sin x}{1+y^2} \).

12. Solve \( \frac{dy}{dx} = \frac{x^2+x}{y^2} \).

13. Find the general solution of \( x \frac{dy}{dx} = y^2 \).

14. Solve \( \frac{dy}{dx} = xy + x + y + 1 \).

15. Find the solution to \( \frac{dy}{dx} = \frac{\sec^2 x}{\tan y} \).

16. Solve \( \frac{dy}{dx} = \frac{1+y^2}{1+x^2} \) with \( y(0)=1 \).

17. Find the general solution of \( y' = \frac{e^x}{y} \).

18. Solve \( \frac{dy}{dx} = \frac{\cos x}{\sin y} \).

19. Which of the following is an implicit solution to \( \frac{dy}{dx} = \frac{xy+y}{x-xy} \)?

20. Solve \( (x^2+1)\frac{dy}{dx} + y = 0 \).
